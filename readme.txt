A WordPress plugin to generate random sales data, demo customers, and product rankings for WooCommerce. Ideal for testing and demo purposes.

== Description ==
The **Random Sales Simulator Plugin** is designed to help developers and shop owners simulate sales data, create demo customers, and add product ratings for WooCommerce stores. This tool is perfect for testing purposes, client demonstrations, and populating your store with sample data.

### Features
- Generate **random sales** data for your WooCommerce products.
- Install **demo customers** for testing user behavior and features.
- Assign **random product rankings (ratings)** to enhance the store's appearance and testing capabilities.
- Customize the number of products and the range of ratings applied.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the simulator options in the WordPress admin panel.

== Usage ==
1. Go to **Tools > Sales Simulator** to access the plugin's functions.
2. Click on:
   - **Install Demo Customers** to generate random customer data.
   - **Install Sample Products** to simulate sales data for selected products.
   - **Assign Random Rankings** to assign random product ratings between 1 and 5 for 20 random products.

== Frequently Asked Questions ==

= How do I install demo customers? =
Simply click the **Install Demo Customers** button in the plugin settings, and it will generate random customer profiles.

= Can I change the number of rankings assigned? =
Currently, the plugin assigns rankings to 20 random products with up to 40 ratings. This can be customized in the plugin's code.

= Is this plugin safe for use on a live store? =
This plugin is meant for testing and demonstration purposes. It is advised to use it on a staging or development site.
